# trackman
TrackMan — Doppler radar launch monitors and tracking platforms for golf, baseball, football, and US football. Hardware (TrackMan 4, TrackMan iO, V3 stadium, B1 portable) plus the TrackMan Range, TrackMan 360, Virtual Golf 3, and TrackMan Golf App software stack, backed by a Cloud GraphQL API, a Range REST API, the TM4 device WebSocket protocol, and the TrackMan Baseball Data API.
